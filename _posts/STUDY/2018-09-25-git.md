### 소스 코드 관리

* git 저장소 만들기 : git init
* 기존 저장소를 clone하기 : git clone
* 파일의 상태 확인하기 : git status
* 파일을 새로 추적하기 : git add
* 변경 사항 커밋하기 : git commit
* 파일을 삭제하기 : git rm
* 변경된 내용 푸시하기 : git push
* 커밋 수정하기 : git commit - - amend
